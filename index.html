    denml.com — System Architecture

    Objective & Overview

    denml.com was architected to serve as an intelligent text distillation and indexing node, replacing static content syndication with multi-agent research pipelines.

    The AI Multi-Agent Blueprint

    The backend coordinates independent AI workers:

    • Reconnaissance Agent: Monitors feed integrations and structures raw web scraping inputs.
    • Parsing & Extraction Agent: Isolates code blocks, technical parameters, and conceptual nodes using strict structured outputs.
    • Synthesis Agent: Summarizes context, links relevant structural nodes, and formats final markdown content.

    Technical Deep Dive

    • Java Backend: Handled query caching, AWS media bucket routing, and rate-limiting limits for LLM integrations.
    • Infrastructure: Scaled dynamically on AWS, using EC2 boxes and automated pipelines connected directly to Github releases.
    • API Limits: Designed exponential-backoff queues using Java threads to prevent API throttling under high bulk load.

    New Bharat Skills — System Architecture

    Objective & Overview

    Designed as an EdTech accelerator for Digital India, this platform is tailored to students in rural and semi-urban districts. AI is utilized to deliver high-quality custom curricula with zero overhead costs.

    Autonomous Features

    • Personalized Curricular Agent: Analyzes a student's diagnostic test and builds localized reading/exercise sets.
    • Mentoring Agent: Responds to code compile or general conceptual queries in real-time, escalating to humans only if unresolved after 3 interactions.
    • Auto-Assessment Agent: Grades open-ended code prompts, outputting precise scoring metrics and specific corrective tips.

    Backend Blueprint

    • Stack: React UI connected to a Spring Boot microservice layer, communicating with a PostgreSQL storage node storing student progress telemetry.
    • Scale Advantage: The deployment tracks and handles support pipelines for thousands of active students with zero manual moderators, ensuring scalability.
